# Quick Installation Guide

## For the Host Player

### 1. Install BepInEx
- Download from: https://github.com/BepInEx/BepInEx/releases
- Extract to your Among Us folder
- Run Among Us once, close it

### 2. Install Sukuna Mod
- Download `SukunaMod.dll` from Releases
- Place in: `Among Us\BepInEx\plugins\`
- Done!

## For Other Players
✅ **No installation needed!** Just join the game normally.

---

## How to Play

### In-Game Controls (Host Only)

| Action | Key |
|--------|-----|
| Transform into Sukuna | **S** |
| Cleave (AOE Attack) | **C** |
| Slash (Single Target) | **V** |
| Domain Expansion | **X** |

### Requirements
- **You must be the Impostor** to use abilities
- **You must be the Host** for the mod to work

---

## If Mod Doesn't Load

1. **Check BepInEx installation:**
   - Look for `doorstop_config.ini` in Among Us folder
   - If missing, BepInEx isn't installed correctly

2. **Check plugin location:**
   - Make sure `SukunaMod.dll` is in `BepInEx\plugins\`
   - NOT in `BepInEx\plugins\SukunaMod\` subfolder

3. **Check console for errors:**
   - Press `F5` in Among Us to open BepInEx console
   - Look for error messages

4. **Reinstall BepInEx:**
   - Delete entire `BepInEx` folder
   - Re-extract the latest BepInEx version
